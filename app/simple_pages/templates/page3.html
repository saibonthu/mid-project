Pytest is a python based testing framework, which is used to write and execute test codes. In the present days of REST services, pytest is mainly used for API testing even though we can use pytest to write simple to complex tests, i.e., we can write codes to test API, database, UI, etc.

In this Project, I created 5 webpages(1 Homepage with Z layout and 4 information pages with F layouts)

in app <- we have templates, Static

in templates <- we have code for all the web pages

in static <- we have .css, images, java script

docker-compose.yml <- Orchestration file that contains the configuration to develop locally, it overrides the Dockerfile to run the flask development server instead of running the gunicorn server that is used for hosting on Heroku

Dockerfile <-File used to create an image to run in a container that runs the program
